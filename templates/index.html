Manual de utilização do módulo financeiro

  • Cadastrar novo setor URL

    • Através do metodo POST informe qual sera o host e a porta do servidor que fornecerá os dados do setor, além do id do setor desejado.

    Exemplo:
    curl -i -H "Accept: application/json" -X POST -d "host=localhost&porta=8000&setor=2" http://localhost:8000/api/setores



  • Criar novo pedido URL

    • Através do metodo POST informe id do setor desejado e descrição a que o pedido se refere.

    Exemplo:
    curl -i -H "Accept: application/json" -X POST -d "coordenacao=1&descricao=pagamento bolsas" http://localhost:8000/api/pedidos



  • Criar novo orçamento URL

    • Através do metodo POST informe id do pedido desejado e valor total que possui o orçamento.

    Exemplo:
    curl -i -H "Accept: application/json" -X POST -d "pedido=1&valor_total=12000" http://localhost:8000/api/orcamentos



  • Criar empenho URL

    • Através do metodo POST informe id do orcamento desejado e valor que será depositado no empenho.

    Exemplo:
    curl -i -H "Accept: application/json" -X POST -d "orcamento=1&valor=1200" http://localhost:8000/api/empenhos
